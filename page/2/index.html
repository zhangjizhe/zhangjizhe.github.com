阿里云配置 nginx https 总结

  • 安装位置太多额，导致重新安装
  • 命令不熟悉
    • nginx -t 检测配置状况
    • nginx -s reload 重启
    • 路径 /etc/nginx/
  • ftp 工具 FileZilla
  • 阿里云公网 IP 更换地址，改为安全组查看，入口挺深的
  • 深坑一点就是 1.18 版本的。d 目录 default 80 端口文件配置，会被覆盖，找半天，才自己看代码注释掉就好了，百度一堆不靠谱的答案。..
  • 另一个马虎点，就是配置证书一直以为 ssl 问题，结果是自己注释习惯性用了//, 应该用#
  • apt-get 安装软件
  • 服务器 git 密码 a19920415
